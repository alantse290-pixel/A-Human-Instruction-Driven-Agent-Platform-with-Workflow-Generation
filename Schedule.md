# Project Schedule

- **Weeks 1-2:** Determine tech stack, set up frontend and backend project skeletons, and test LLM API calls.

- **Weeks 3-4:** Design workflow JSON schema, develop prompt engineering for workflow generation, and generate valid workflow JSON from natural language.

- **Weeks 5-7:** Develop React Flow visual editor, and implement basic node dragging, connecting, and configuration panels.

- **Weeks 8-9:** Implement workflow execution engine, and run end-to-end process (user input → generate workflow → display → execute → return results).

- **Weeks 10-12:** Develop two specific applications (literature retrieval and campus policy Q&A), and handle specific logic for each agent.

- **Weeks 13-15:** Conduct evaluation experiments (accuracy and usability testing), fix bugs, and write the final report.
